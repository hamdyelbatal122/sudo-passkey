# Security Policy

## Supported Versions

The latest minor release is supported.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

1. Email maintainers privately with details
2. Include reproduction steps and impact
3. Allow reasonable disclosure time before public release

## Security recommendations

- Use least-privilege sudoers rules
- Keep OS and browser updated
- Prefer hardware-backed passkeys
- Rotate and revoke compromised credentials
