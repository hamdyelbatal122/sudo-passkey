# Helio — Website Template

A modern, responsive **website template** built with HTML, CSS, and JavaScript. Clean layout, smooth design, and easy to customize for personal or business use.

## ✨ Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive across all devices
- ⚡ Smooth animations and transitions
- 🔧 Easy to customize and extend
- 🌐 Cross-browser compatible
- 📦 Zero external dependencies

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamdyelbatal122/helio.git
   cd helio
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```

## 📁 Project Structure

```
helio/
├── index.html        # Main HTML file
├── css/
│   └── style.css     # Main stylesheet
├── js/
│   └── main.js       # JavaScript logic
├── images/           # Image assets
└── README.md
```

## 🎨 Customization

1. Edit `index.html` to change content and structure
2. Modify `css/style.css` to update colors and fonts
3. Update `js/main.js` for any interactive behavior
